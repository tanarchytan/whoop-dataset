# Code of Conduct

Be respectful and act in good faith. This project follows the spirit of the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In particular:

- **Only contribute data about yourself, with your consent.** Never submit anyone else's data.
- **Respect privacy.** Do not include or attempt to de-anonymise personal information. Sharing others'
  private information is not tolerated.
- Harassment and discrimination of any kind are not welcome.

Report concerns by opening a GitHub issue, or privately to the maintainer. See
[SECURITY.md](SECURITY.md) for the sensitive-data reporting path.
